Font pour l'hébreu.

Pour améliorer l'affichage des textes hébreux, nous vous conseillons d'installer la font Frank Ruhl Libre.

(haut)

Modules.

Theke fonctionne à partir de modules. Chaque module peut contenir une version de la bible, un dictionnaire, ou encore un ouvrage. Pour l'instant, Theke ne sait pas encore télécharger et installer les modules Sword. Vous devez donc passer par un logiciel tiers pour le faire, par exemple Xiphos.

Dans Xiphos, ouvrez le gestionnaire de modules dans Édition > Gérer les modules. La colonne de gauche présente deux sections : dépôts et modules. Dans Dépôts > Sélection, vous pouvez choisir un dépôt distant d'où vous pourrez télécharger des modules.

Une fois que vous avez sélectionné un dépôt distant, vous pouvez consulter les modules qu'il contient et installer ceux qui vous intéressent à partir de Modules > Installer/Maj.

Vous trouverez ci-dessous une liste de quelques modules que nous vous recommandons pour commencer à utiliser Theke.

Nom Dépôt Catégorie Description
MorphGNT CrossWire Textes bibliques > Κοινὴ Ἑλληνική Morphologically Parsed Greek New Testament based on the SBLGNT.

Morphologie.

Le décryptage des informations morphologiques n'est pas encore complet. En attendant, il est possible de consulter les références suivantes :

  • oshm − codes morphologiques du module OSHB ;
  • robinson − codes morphologiques du module MorphoGNT.

Ouvrir un document.

La GotoBar, en haut à droite de la fenêtre, vous permet d'ouvrir n'importe lequel des documents de Theke. Pour cela, commencez par taper les premières lettres du titre du livre biblique ou du document que vous souhaitez consulter.
